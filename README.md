## 📝 Description

**F.R.I.D.A.Y.** is a powerful Python-based virtual assistant inspired by Google Assistant and Amazon Alexa. Designed for voice-controlled operation, it can:

- Open popular websites like Google, YouTube, LinkedIn, Facebook, and more.
- Play music directly from YouTube.
- Fetch and read out the latest news using the NewsAPI.
- Answer your questions using ChatGPT API integration.

This assistant is designed to be modular and customizable, allowing you to add new functionalities with ease.

## 🚀 Installation Instructions

1. **Clone this repository** to your local machine.

   ```bash
   git clone https://github.com/yourusername/friday-virtual-assistant.git
   cd friday-virtual-assistant
   ```

2. **Set up a virtual environment**:

   ```bash
   python -m venv .env
   source .env/bin/activate  # For Linux/macOS
   .env\Scripts\activate     # For Windows
   ```

3. **Install the dependencies**:

   ```bash
   pip install -r requirements.txt
   ```

4. **Add your API Keys**:
   - Replace placeholders in the code with your actual API keys (e.g., NewsAPI, OpenAI).
   - Make sure these keys are valid and obtained from authenticated sources.

## 🎤 Usage

- **To initiate the assistant**, use the command:  
  `"hello"`  
  This will activate F.R.I.D.A.Y.

- **To open a website**, use:  
  `"open [website name]"`  
  Example: `"open youtube"`

- **To play music**, simply use a music command, and it will fetch and play music from YouTube.

- **To get news headlines**, use:  
  `"headlines"`  
  This will fetch the latest news using the NewsAPI.

- **To ask a question**, begin with `"what"`, `"who"`, `"where"`, etc.  
  This triggers the ChatGPT API to fetch and deliver a response.
